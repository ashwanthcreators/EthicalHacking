COMMANDS




Download Osintgram
 
  
 git clone https://github.com/Datalux/Osintgram.git

Make sure you have Python 3.6+ and pip3 Installed
 
         
  sudo apt install python3

         
 sudo apt install python3-pip

Install requirements (make sure you are in the Osintgram directory)
 
 
  pip3 install -r requirements.txt

Create Login Info
 
  
mkdir config

         
cd config 

         
  echo "username" > username.conf

         
echo "password" > pw.conf

         
echo "{}" > settings.json

Start Exploring (make sure you are back in the Osintgram directory)
 

cd .. 

         
 python3 main.py instagram_target

Now you will find a error like this (Error: "username" field cannot be blank in "config/credentials.ini")

Follow these steps to fix the error 
  
cd config 

         
nano credentials.ini

there you will username and password

write your username and password next to = without gap

save it by using this command ctrl+s

exit by typing ctrl+x

next typing (cd..)

Start Hacking (make sure you are back in the Osintgram directory)
 
 
python3 main.py instagram_target
