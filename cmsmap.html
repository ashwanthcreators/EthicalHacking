Steps


Installation
You can download the latest version of CMSmap by cloning the GitHub repository:

git clone https://github.com/Dionach/CMSmap

If you would like to run cmsmap from anywhere in your system you can install it with pip3 :


cd CMSmap

pip3 install .

To uninstall it :

pip3 uninstall cmsmap -y

usage :
python3 cmsmap.py https://google.com -t 500

https://google.com should be replaced by your victims website
